# Naphattha Portfolio Website

A Personal Portfolio Built with Flutter for Web
This project showcases the personal portfolio of Naphattha, beautifully crafted using the Flutter framework for a fast, responsive, and cross-platform web experience. It serves as a professional online presence to display projects, skills, and contact information.

**Live Demo:** [https://naphattha.github.io/flutter_portfolio_website/](https://naphattha.github.io/flutter_portfolio_website/)

## Features
- **Responsive Design:** Supports mobile, tablet, and desktop layouts.
- **Dynamic Projects:** Interactive filtering for Web, Mobile, and other categories.
- **Modern UI:** Smooth animations and hover effects using Flutter's core engine.

## Tech Stack & Tools

- **Framework:** Flutter Web
- **Language:** Dart
- **Design Pattern:** Clean Architecture & Responsive Design
- **Key Packages:**
  - `font_awesome_flutter` (Iconography)
  - `google_fonts` (Custom Typography)
  - `url_launcher` (External links & contact)
  - `simple_gradient_text` (Dynamic UI effects)

## Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/naphattha/flutter_portfolio_website.git](https://github.com/naphattha/flutter_portfolio_website.git)
   ```
2. Get dependencies:
   ```bash
   flutter pub get
   ```
3. Run the project:
   ```bash
   flutter run -d chrome
   ```
